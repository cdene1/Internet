What is hypertext transfer protocol and the domain name system?




Hypertext tranfer protocol:

Probelem it solves:

It allows computers to request and share webpages, images, videos, and other file types on the internet


How does it work:

An HTTP "GET" request is the way computers communicate through the Internet as part of a TCP packet to ask for the information they need, to load a website, open an image, etc. The servers that receive the request respond with the files requested which are then displayed on your browser.






Domain name system:

Problem it solves:

Translate human-readable domains like wikipedia.com into IP addresses that the internet can use


How does it work:

The DNS works like a phonebook. When you try to go to a domain like wikipedia.com your computer contacts a system of servers collectively called DNS which keeps track of the IP addresses associated with each domain name. Your computer is now able to open the domain associated with the IP address stored in the DNS.
